# Projekt Shiny Pogoda R
Projekt Wykonali:
Wojtek Szmidt, Sebastian Zgliński, Daria Sieg i Jakub Zamczyk
## Celem tego projektu jest wizualizacja zmiany temperatury, ciśnienia oraz opadów na przekroju roku. 
Link do strony: [Projekt Shiny](https://z6rhh0-jakub-zamczyk.shinyapps.io/Projekt_Temperatura/)

Można dowolnie filtrować dane wybierając miesiące oraz rysować linię trendu miesięcznie czy ogólnie.<br>
